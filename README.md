#  Курсовой проект (2 курс 3 семестр)

## ПРИМЕНЕНИЕ ГЕНЕТИЧЕСКИХ АЛГОРИТМОВ ДЛЯ ОБУЧЕНИЯ НЕЙРОННЫХ СЕТЕЙ

Результатом выполнения настоящей курсовой работы является программа «Agent 007». 
Суть данной программы заключается в том, что агент, контролируемый при помощи нейронной сети, должен избегать контакта с противниками, как можно более длительное время.

### Цель и задачи

**Целью** данной курсовой работы является подробное изучение фундаментальных математических основ генетических алгоритмов, реализовать программу для обучения нейронной сети с помощью генетического алгоритма, и на показать практическое применение знаний, навыков и умений, полученных при изучении дисциплины «Технологии и методы программирования».

Для достижения поставленной цели были выделены следующие **задачи**:

1. Изучить материалы, посвещенные генетическим алгоритмам;
2. Изучить материалы, посвещенные нейронным сетям;
3. Ознакомиться со способом обучения нейронной сети;
4. Разработать программу для демонстрации применения генетических алгоритмов для обучения нейронной сети
